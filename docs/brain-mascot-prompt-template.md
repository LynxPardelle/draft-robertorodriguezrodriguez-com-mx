# Brain Mascot Prompt Template

Use this template to create more Roberto Rodriguez Rodriguez brain mascot stickers without using Roberto's face or likeness.

## Prompt

Create a friendly 2D cartoon pink brain mascot sticker for a Spanish online psychotherapy website. The mascot should feel professional, calm, wholesome, warm, and emotionally safe. It should be a cute brain character with simple expressive eyes, soft rounded shapes, subtle highlights, and clean sticker-style edges.

Pose/context: `{pose_or_action}`.

Optional prop: `{prop_or_symbol}`.

Intended section: `{site_section_or_message}`.

Style requirements:
- Soft pink brain mascot, not a human portrait.
- No resemblance to any real person.
- No text, no letters, no logo, no watermark.
- Clean front-facing or three-quarter view.
- Generous padding around the character so hands, legs, props, and outline are not clipped.
- Flat chroma-key green background exactly `#00ff00`.
- Do not use green in the subject, prop, outline, highlights, or shadows.
- No background scene.

Negative prompt:
No human face, no realistic anatomy, no medical gore, no anxiety-inducing expression, no sadness as the final mood, no text, no watermark, no cropped limbs, no cut-off prop, no green subject pixels.

## Post-Processing Checklist

1. Remove the chroma-key background with a soft matte.
2. Export transparent PNG at `420x420`.
3. Keep file size small enough for web use.
4. Upload as a shared draft asset under `shared/mascot-images`.
5. Use positive CSS offsets and content-width positioning so the sticker does not get clipped.
